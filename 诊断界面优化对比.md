## 动画效果清单

### 新增动画

1. **slideIn** - 消息滑入动画
   - 从下方12px滑入
   - 同时淡入显示
   - 使用 cubic-bezier 缓动

2. **pulse** - 工具运行状态脉冲
   - 透明度在1和0.6之间循环
   - 2秒循环周期
   - 无限循环

3. **blink** - 加载指示器闪烁
   - 点状指示器闪烁
   - 1.4秒循环周期

### 优化的过渡效果

1. **会话项过渡**
   - 使用 cubic-bezier(0.4, 0, 0.2, 1)
   - 0.2秒过渡时间
   - 包含背景、颜色、位移

2. **按钮过渡**
   - 使用 cubic-bezier(0.4, 0, 0.2, 1)
   - 0.2秒过渡时间
   - 包含背景、缩放、阴影

3. **面板切换**
   - 使用 cubic-bezier(0.4, 0, 0.2, 1)
   - 0.3秒过渡时间
   - 平滑的宽度变化

## 颜色和视觉层次

### 强调色使用

- **蓝色 (--accent-blue)**: 主要操作、活动状态
- **紫色 (--accent-purple)**: AI助手标识
- **青色 (--accent-cyan)**: 工具调用标识
- **绿色 (--accent-green)**: 成功状态
- **红色 (--accent-red)**: 危险操作、错误状态

### 阴影层次

1. **轻微阴影**: 消息气泡、卡片
2. **中等阴影**: 悬停状态、工具面板
3. **强阴影**: 按钮悬停、模态框

## 响应式断点

- **1200px**: 工具面板宽度调整为320px
- **768px**: 移动端布局，消息宽度95%，工具面板绝对定位

## 性能优化措施

1. **使用 transform 和 opacity**
   - 触发GPU硬件加速
   - 避免重排和重绘

2. **选择性过渡**
   - 只对必要属性应用过渡
   - 避免 all 过渡

3. **合理的动画时长**
   - 快速反馈: 0.2秒
   - 状态变化: 0.3秒
   - 避免过长动画影响体验

## 无障碍改进

1. **键盘导航**
   - 所有交互元素可通过Tab访问
   - 清晰的焦点指示器
   - 使用 focus-visible 优化

2. **颜色对比度**
   - 保持足够的文本对比度
   - 状态指示不仅依赖颜色

3. **动画控制**
   - 尊重用户的动画偏好设置
   - 可通过媒体查询禁用动画
