# 更新日志

本文档记录 DBClaw 项目的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.9.3] - 2026-04-22

### 新增
- 新增系统时间查询技能
- 优化 AI 对话工具卡片样式
- 增强诊断事件持久化容错性
- 补充软删除字段迁移

### 修复
- 修复 SAP HANA 执行计划清理中的 SQL 注入漏洞
- 修复多处空异常捕获导致的调试困难问题
- 移除生产环境中的调试 print 语句
- 统一版本号管理

### 变更
- 统一查询执行接口并优化前后端交互
- 重构数据模型和迁移脚本，统一命名规范
- 移除技能授权调试日志

### 安全
- 修复 HANA 服务中的 SQL 注入漏洞（使用参数化查询）
- 改进异常处理，避免捕获所有异常类型
- 清理调试代码，避免敏感信息泄露

## [0.9.2] - 2026-04-14

### 新增
- 支持 SAP HANA 数据库
- 新增主机配置缓存功能
- 新增告警事件追踪
- 新增集成模板系统

### 修复
- 修复时区相关的数据一致性问题
- 修复告警聚合逻辑
- 修复 SQL Server 2012 兼容性问题

### 变更
- 优化指标采集性能
- 改进巡检报告生成逻辑
- 重构通知分发器

## [0.9.1] - 2026-03-20

### 新增
- 支持 openGauss 数据库
- 新增 AI 告警策略
- 新增基线检测功能
- 新增飞书、钉钉、企业微信机器人集成

### 修复
- 修复 PostgreSQL 连接池泄漏问题
- 修复 Oracle 11g 兼容性问题
- 修复会话序列化问题

### 变更
- 优化知识库检索性能
- 改进 AI 对话上下文管理
- 重构技能执行引擎

## [0.9.0] - 2026-02-28

### 新增
- 首个公开版本
- 支持 MySQL、PostgreSQL、Oracle、SQL Server、TDSQL-C MySQL
- 数据源统一管理
- 主动监控和告警
- 自动巡检
- AI 智能诊断
- 技能系统
- 知识增强
- 外部集成

### 安全
- 实现加密密钥管理
- 实现启动自检机制
- 实现会话管理
- 实现权限控制

- **新增**: 新功能

- **性能**: 性能优化
